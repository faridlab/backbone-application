# ADR-0002: Schema YAML as the single source of truth

- **Status:** Accepted
- **Deciders:** Backbone maintainers
- **Applies to:** domain `module` projects; regeneration behavior visible in this service

## Context

A domain concept — an entity, its fields, its persistence, its CRUD surface, its migrations, its
DTOs — has to stay consistent across many artifacts. Maintaining those by hand guarantees drift: a
column added to a migration but not the entity, a DTO that forgets a field, a service alias that
doesn't match the repository. Multiplied across many modules and many services composing them, the
drift becomes unmanageable.

## Decision

In a `module` project, **`schema/models/*.model.yaml` is the single source of truth.** Entities,
migrations, CRUD wiring, and DTOs are *generated* from it by `metaphor schema generate`. Hand-written
code is the explicit exception, protected by three deterministic mechanisms:

1. `metaphor.codegen.yaml` → `user_owned:` globs — files skipped wholesale on regen.
2. `// <<< CUSTOM … // END CUSTOM` markers — regions preserved inside generator-emitted files.
3. `-- Generated by metaphor-schema` migration headers — headerless files survive the sweep.

## Consequences

**Positive**

- One edit (the schema) propagates consistently to every generated artifact.
- Drift between entity, migration, and DTO is structurally prevented.
- Regeneration is safe and repeatable because hand-written code announces itself.

**Negative / costs**

- Contributors must learn the regen-safety contract; a hand-written file in a generator tree with no
  `user_owned:` entry is silently deleted on the next `--force`.
- The generator becomes a critical dependency; a generator bug affects every module.
- Edits outside `// <<< CUSTOM` markers in generated files are clobbered — a real footgun until learned.

## Consequences for *this* service

This is a `backend-service`, not a module — it holds no schema models of its own (only
`schema/models/sysparam.model.yaml` ships as a minimal example). But it inherits the contract: any
hand-written file added inside a generator-owned tree here must be listed under `user_owned:` in
[`metaphor.codegen.yaml`](../../metaphor.codegen.yaml). See [maintainer guide](../maintainer-guide.md#the-regeneration-model-know-this-cold).
